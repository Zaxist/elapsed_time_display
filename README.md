# timer_text

A self updating Timer Text widget for flutter.
## Getting Started
include in your pubspec yaml file like 
```
timer_text:
    git:
        url: https://github.com/Zaxist/timer_text.git
```
      
then import 
`import 'package:timer_text/timer_text.dart';`

and use like 
`TimerText(startTime: DateTime.now(),)`
